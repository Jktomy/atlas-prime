# Prime Change Routes

| Concept | Meaning |
|---|---|
| Authorizer | Jayson or one exact delegated authority; controls scope and permanence |
| Operator | Athena, Jayson, or another named actor performing the bounded action |
| Delivery route | Spear for direct delivery; Arrow/Bow for delegated immutable delivery |
| Independent local route | Sword sealed package using the Oathbringer method |
| Direct bounded method | Phoenix Blade |
| Safe rerouting | Aegis Break |
| Hosted or local launcher | Thin intake that invokes an engine without becoming one |
| Normal repository engine | Prime Thread Engine, singular |
| Repository substrate | Fresh Clone First |
| AI-assisted work surface | Shardplate |
| Execution environment | Stormlight |

```text
Spear -> Thread Engine
Arrow -> Bow -> Thread Engine
Sword -> Oathbringer -> PowerShell
Aegis Break -> Phoenix Blade
```

Aegis Break selects a safe equivalent route when the normal path is obstructed. It does not erase Aegis or grant new authority.

Every route:

- separates Build from Execute;
- uses exact base and payload locks;
- stops at draft PR for Noctua;
- prohibits direct main and force push;
- prohibits automatic ready and automatic merge;
- requires separate Jayson permanence authority.

Prime has one normal repository engine. Hosted Spear and local launchers may validate, compile, and invoke exact missions, but they do not become additional repository writers. Thread Engine never performs its own self-change.
