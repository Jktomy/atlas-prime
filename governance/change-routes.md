# Prime Change Routes

```text
Spear → Thread Engine
Arrow → Bow → Thread Engine
Sword → Oathbringer → PowerShell
```

Aegis Break selects a safe equivalent route when the normal path is obstructed. It does not erase Aegis or grant new authority.

Every route:

- separates Build from Execute;
- uses exact base and payload locks;
- stops at draft PR for Noctua;
- prohibits direct main and force push;
- prohibits automatic ready and automatic merge;
- requires separate Jayson permanence authority.
