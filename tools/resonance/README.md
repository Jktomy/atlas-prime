# Resonance reconciler

`reconcile_findings` groups independently sealed findings by semantic claim key,
preserves every finding digest, and classifies exact statement agreement,
conflict, or a single-lane novel claim. Provider and model labels never vote.

Athena Refraction produces a non-final recommendation for the Aberration
Register. It cannot accept, reject, merge, or promote a finding. Local-model
findings remain excluded until an independent runtime-proof digest exists.
